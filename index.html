Sobre mim

Olá! Meu nome é Kauã Chaves e tenho uma profunda paixão por Desenvolvimento em geral (Web, Desktop, Mobile), entretanto meu “xodó” é o Desenvolvimento Web, pois é onde já possuo uma boa familiaridade. Meu interesse por desenvolver para a web começou em 2020, onde tive meus primeiros contatos com Html e Css com o saudoso Prof. Guanabara.


Atualmente tenho avançado na área com a linguagem Javascript, onde venho utilizando os frameworks Node.js e React.js para estruturalizar todo o back-end e Front-end do site/sistema web, procurando entregar uma aplicação funcional, completa, responsiva, usual e útil para soluções de problemas reais.


Juntamente, estou cursando Análise e Desenvolvimento de Sistemas na UNOPAR (Universidade Norte do Paraná), já cursando o 2º semestre do curso venho adiquirindo vários conhecimentos em Engenharia de Software, Modelagem de Dados, Modelagem de Sistemas, entre outras ciências essenciais da área.
